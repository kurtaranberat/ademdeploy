# BRAND-A Sales Intelligence Dashboard

3 boyutlu, interaktif satış analitik platformu.

## Başlatma

### 1. Backend (Python/Flask)
```
start_backend.bat
```
→ http://localhost:5000

### 2. Frontend (React)
```
start_frontend.bat
```
→ http://localhost:3000

## Gereksinimler
- Python 3.9+
- Node.js 18+

## Özellikler
- 🏠 **Hero** — Canlı KPI özeti + parçacık animasyonu
- 📊 **Genel Bakış** — 2024 aylık hedef vs gerçekleşme grafikleri
- 🏪 **Bayiler** — 22 bayi ısı haritası, sıralama, radar grafik
- 🚗 **Modeller** — Model bazlı trend, pasta grafik, yığılmış bar
- 📦 **Araçlar** — Kanal, şehir, renk, aylık trend analizi
- 🌐 **3D Görünüm** — Three.js ile interaktif 3D bar chart
